LUAL_ADDVALUE(3) Library Functions Manual LUAL_ADDVALUE(3)

NAME

luaL_addvalue — adds the value at the top of the stack to the buffer, function indicator [-1, +0, m]

SYNOPSIS

#include <lauxlib.h>

void
luaL_addvalue(luaL_Buffer *B);

DESCRIPTION

luaL_addvalue() adds the value at the top of the stack to the buffer B (see luaL_Buffer(3)). Pops the value.

This is the only function on string buffers that can (and must) be called with an extra element on the stack, which is the value to be added to the buffer.
