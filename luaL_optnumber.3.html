LUAL_OPTNUMBER(3) Library Functions Manual LUAL_OPTNUMBER(3)

NAME

luaL_optnumber — if the function argument is a number, returns this number, function indicator [-0, +0, v]

SYNOPSIS

#include <lauxlib.h>

lua_Number
luaL_optnumber(lua_State *L, int narg, lua_Number d);

DESCRIPTION

luaL_optnumber() if the function argument narg is a number, returns this number. If this argument is absent or is nil, returns d. Otherwise, raises an error.
