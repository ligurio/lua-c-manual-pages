LUAL_DOSTRING(3) Library Functions Manual LUAL_DOSTRING(3)

NAME

luaL_dostring — loads and runs the given string, function indicator [-0, +?, m]

SYNOPSIS

#include <lauxlib.h>

int
luaL_dostring(lua_State *L, const char *str);

DESCRIPTION

luaL_dostring() loads and runs the given string. It is defined as the following macro:

(luaL_loadstring(L, str) || lua_pcall(L, 0, LUA_MULTRET, 0))

RETURN VALUES

It returns 0 if there are no errors or 1 in case of errors.
