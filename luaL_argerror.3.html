LUAL_ARGERROR(3) Library Functions Manual LUAL_ARGERROR(3)

NAME

luaL_argerror — raises an error with the message, function indicator [-0, +0, v]

SYNOPSIS

#include <lauxlib.h>

int
luaL_argerror(lua_State *L, int narg, const char *extramsg);

DESCRIPTION

luaL_argerror() raises an error with the following message, where func is retrieved from the call stack:

bad argument #<narg> to <func> (<extramsg>)

This function never returns, but it is an idiom to use it in C functions as return luaL_argerror(args).
