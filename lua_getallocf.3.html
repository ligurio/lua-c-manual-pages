LUA_GETALLOCF(3) Library Functions Manual LUA_GETALLOCF(3)

NAME

lua_getallocf — returns the memory-allocation function of a given state, function indicator [-0, +0, -]

SYNOPSIS

#include <lua.h>

lua_Alloc
lua_getallocf(lua_State *L, void **ud);

DESCRIPTION

lua_getallocf() returns the memory-allocation function of a given state.

RETURN VALUES

If ud is not NULL, Lua stores in *ud the opaque pointer passed to lua_newstate(3).
