LUA_SETALLOCF(3) Library Functions Manual LUA_SETALLOCF(3)

NAME

lua_setallocf — changes the allocator function, function indicator [-0, +0, -]

SYNOPSIS

#include <lua.h>

void
lua_setallocf(lua_State *L, lua_Alloc f, void *ud);

DESCRIPTION

lua_setallocf() changes the allocator function of a given state to f with user data ud.

Roberto Ierusalimschy, Luiz Henrique de Figueiredo, and Waldemar Celes, Lua 5.1 Reference Manual.
