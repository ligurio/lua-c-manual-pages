NAME

lua_istable — check whether a value is a table, function indicator [-0, +0, -]

SYNOPSIS

#include <lua.h>

int
lua_istable(lua_State *L, int index);

DESCRIPTION

lua_istable()

RETURN VALUES

Returns 1 if the value at the given acceptable index is a table, and 0 otherwise.
