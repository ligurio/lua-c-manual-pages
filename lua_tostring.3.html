LUA_TOSTRING(3) Library Functions Manual LUA_TOSTRING(3)

NAME

lua_tostring — converts the Lua value to a C string, function indicator [-0, +0, m]

SYNOPSIS

#include <lua.h>

const char *
lua_tostring(lua_State *L, int index);

DESCRIPTION

lua_tostring() converts the Lua value at the given acceptable index to a C string.

Equivalent to lua_tolstring(3) with len equal to NULL.
