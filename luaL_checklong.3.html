LUAL_CHECKLONG(3) Library Functions Manual LUAL_CHECKLONG(3)

NAME

luaL_checklong — checks whether the function argument is a number, function indicator [-0, +0, v]

SYNOPSIS

#include <lauxlib.h>

long
luaL_checklong(lua_State *L, int narg);

DESCRIPTION

luaL_checklong() checks whether the function argument narg is a number and returns this number cast to a long.
