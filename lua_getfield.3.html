LUA_GETFIELD(3) Library Functions Manual LUA_GETFIELD(3)

NAME

lua_getfield — pushes onto the stack the value t[k], where t is the value at the given valid index, function indicator [-0, +1, e]

SYNOPSIS

#include <lua.h>

void
lua_getfield(lua_State *L, int index, const char *k);

DESCRIPTION

lua_getfield() pushes onto the stack the value t[k], where t is the value at the given valid index. As in Lua, this function may trigger a metamethod for the "index" event.
