LUAL_GSUB(3) Library Functions Manual LUAL_GSUB(3)

NAME

luaL_gsub — creates a copy of string A by replacing any occurrence of the string B with the string C, function indicator [-0, +1, m]

SYNOPSIS

#include <lauxlib.h>

const char *
luaL_gsub(lua_State *L, const char *s, const char *p, const char *r);

DESCRIPTION

luaL_gsub() creates a copy of string s by replacing any occurrence of the string p with the string r. Pushes the resulting string on the stack and returns it.
