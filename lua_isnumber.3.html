LUA_ISNUMBER(3) Library Functions Manual LUA_ISNUMBER(3)

NAME

lua_isnumber — check whether a value is a number, function indicator [-0, +0, -]

SYNOPSIS

#include <lua.h>

int
lua_isnumber(lua_State *L, int index);

DESCRIPTION

lua_isnumber() returns 1 if the value at the given acceptable index is a number or a string convertible to a number, and 0 otherwise.

RETURN VALUES

Returns 1 if the value at the given acceptable index is a number or a string convertible to a number, and 0 otherwise.
